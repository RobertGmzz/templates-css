14 de Febrero

El objetivo principal de CSS Switch Gallery es ofrecer una variedad de estilos predefinidos a desarrolladores que necesitan una guía visual rápida para su sitio web antes de crear o personalizar sus propios estilos CSS. También está pensado para quienes simplemente buscan un diseño rápido para una página estática, como por ejemplo un portafolio, landing, etc. Ten en cuenta que estos estilos no están pensados para proyectos que ya estén muy avanzados.

Para la demo, me basé en la referencia de elementos de MDN.

Listas, Ordenada y Desordenada

Características

  • Se puede usar directamente, sin necesidad de instalación.
  • Puedes descargar la plantilla y modificarla según lo necesites.
  • Incluye algunas clases opcionales para facilitar un uso rápido.

Como usar

  1. Cambia los estilos del demo en tiempo real usando los botones switch y elige el que más te guste.
  2. Copia el elemento <link> que se encuentra sobre el visualizador.
  3. Pégalo dentro del <head> de tu HTML.
Aquí tienes el texto mejorado, más claro y con mejor fluidez: Clases? Sí, incluí clases opcionales en las plantillas para que puedas utilizarlas rápidamente si lo necesitas. La idea es similar a la forma en que se implementan las clases en Tailwind. A lo largo del demo encontrarás ejemplos junto con su sintaxis.

Clases para botones

En muchas páginas web, los botones tienen estilos distintos según su intención y nivel de importancia. Aquí encontrarás clases para botones que te permitirán ampliar y enriquecer el diseño de tus interfaces de usuario.

Utiliza las clases:

button-primary
button-secondary
button-tertiary
button-disabled
button-fit

No te dejes llevar por el nombre de las clases. Ningún botón es semánticamente más importante que otro; simplemente les di esos nombres para diferenciarlos y facilitar su uso. Utilízalos según lo consideres más conveniente.

Clases para diferenciar secciones de página

Finalmente, aquí tienes algunas clases para contenedores. Es común que muchas páginas web utilicen distintos colores para determinadas secciones de la página o para elementos que necesitan destacar visualmente. Al igual que con los botones, dales el uso que más te convenga.
